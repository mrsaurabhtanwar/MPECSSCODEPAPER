# Verification Directory

This directory is used for storing "Gold Standard" or reference results.

## How it differs from the `results/` folder:

- **`results/`**: This is where your daily work is saved. Every time you run a test, the results go here.
- **`verification/`**: This is for results that we know are 100% correct. We use these files to compare against new versions of the solver to make sure we haven't broken anything.

Currently, this folder is empty because we are starting fresh with the
latest archived version of MPECSS (v1.0.3). Once we have a new
"Official" set of results, we will move the best files here for safe
keeping.
